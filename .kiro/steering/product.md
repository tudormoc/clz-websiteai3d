# Product Overview

CLZ (Cooperativa Lavoratori Zanardi) is a premium bookbinding company website showcasing their industrial artisan services. The site serves as a digital portfolio and inquiry platform for museums, art galleries, and luxury publishers.

## Core Purpose
- Present CLZ's specialized hardcover/cardboard binding services ("Cartonato")
- Showcase maxi format capabilities (up to 435mm x 605mm)
- Display portfolio of prestigious client work
- Enable project inquiries via contact form

## Target Audience
- Museums and cultural institutions
- Art galleries
- Luxury publishers
- Architecture firms requiring high-end publications

## Key Features
- Multi-language support (Italian, English, French)
- Interactive 3D book visualizations
- Binding process demonstrations
- Format configurator tool
- Project portfolio with filtering
